# Можно ли использовать JPA c noSQl базами данных?
> 
Можно, но нужно быть готовым к тому что многие функции из JPA будут бесполезны для nosql баз данных.

Существуют несколько реализаций JPA для nosql:  
* DataNucleus
* Kundera
* ObjectDB и другие.
